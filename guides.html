Minecraft Resource Packs


Windows (Java Edition)

  • Download the resource pack you want
  • Copy the file you downloaded
  • Press Win+R. A window titled "Run" should appear
  • In the textbox inside the "Run" window, type %appdata% and click "Ok"
  • In the folder that just appeared, open the ".minecraft" folder
  • In the ".minecraft" folder, open the "resourcepacks" folder
  • Paste the resource pack in this folder
  • Open Minecraft, select "Options", then "Resource Packs", select your resource pack and select "done"

MacOS

  • Download a resource pack of your choice
  • Copy the file you downloaded
  • Open your "minecraft" folder. (Should be at ~/Library/Application Support/minecraft)
  • Open the "resourcepacks" folder
  • Paste the resource pack in this folder
  • Open Minecraft, select "Options", then "Resource Packs", select your resource pack and select "done"

To access ~/Library/ on a Mac running MacOS 10.7 and above, switch to Finder, open the 'Go' Menu, and while holding down Option, select 'Library'

Linux

  • Download a resource pack of your choice
  • Copy the file you downloaded
  • Open your "minecraft" folder. (Should be at ~/.minecraft)
  • In the "minecraft" folder, open the "resourcepack" folder
  • Paste the resource pack in this folder
  • Open Minecraft, select "Options", then "Resource Packs", select your resource pack and select "done"

Minecraft Datapacks


Windows (Java Edition)

  • Download the datapack you want
  • Copy the file you downloaded
  • Press Win+R. A window titled "Run" should appear
  • In the textbox inside the "Run" window, type %appdata% and click "Ok"
  • In the folder that just appeared, open the ".minecraft" folder
  • In the ".minecraft" folder, open the "saves" folder
  • Locate the folder for your world and open it
  • Open the "datapacks" folder
  • Paste the datapack in this folder
  • You may need to leave and rejoin your world, or run /reload.

MacOS
